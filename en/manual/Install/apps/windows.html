Windows

Download

Please go to the download page of Windows installer of swEOS to download the proper version (e.g. windows 10) according to your system version.

Install

The downloaded installer is a .exe file. Then double click it to install just like any other general windows software installer, there is nothing special. After successfully installing, one can get something like Fig. 2.

Fig. 2 Snapshot of swEOS desktop app installed in windows 10 system.

Desktop app with GUI

If you want to use the GUI version of swEOS, just click the shortcut automatically added in the start menu (see Fig. 2).

Command line tool

The standalone command line tool(cmd) is also included in the installer, if you want to use this cmd app, just need to add the command line tool install path (see Fig. 2) to the system environment variable of PATH (see Fig. 3). Then swEOS command can be used in PowerShell or traditional cmd terminal.

Fig. 3 Snapshot of environment variable setting.
